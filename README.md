Corvisa Integration inside Codebox
===============


### Configuration (ENV)

| Name | Description |
| ---- | ----------- |
| `CORVISA_APPID` | Application ID  of the app on Corvisa |
| `CORVISA_APIKEY` | API key |
| `CORVISA_ENDPOINT` | Endpoint for the corvisa endpoint (default is `https://api.us1.corvisa.io`) |
